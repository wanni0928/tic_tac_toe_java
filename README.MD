# Tic Tac Toe project
## Project summary
1. Board
 - 3 x 3
2. Player
 - human player
 - AI player
3. Winner Judgement
 - case01. 3 stones horizontally
 - case02. 3 stones vertically
 - case03. 3 stones diagonally

## Design
1. Input Integer N
2. Input whether player is AI or not.
3. If one of player get N win, game will terminated.. until then game will continue.
